# PORTFOLIO MASTER PLAN v1.0

## Project Vision

Create a cinematic, interactive portfolio experience inspired by premium product websites, modern storytelling experiences, and advanced motion design.

The goal is NOT to copy any reference.

The goal is to combine:

* Premium 3D presentation
* Scroll storytelling
* Strong project showcase
* Smooth page transitions
* Unique visual identity

into a portfolio that feels personal and memorable.

# Design Rules

1. Dark theme first

2. Premium over flashy

3. Motion must support storytelling

4. Every animation must have purpose

5. Easy future replacement of 3D assets

6. Mobile experience must remain smooth
